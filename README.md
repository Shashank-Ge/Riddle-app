# Riddle App

A simple React app that shows riddles.

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/Shashank-Ge/Riddle-app.git

2. Navigate to the project directory:
```bash
cd Riddle-app
3. Install dependencies:
```bash
npm install
4. Start the development server:
```bash
npm start
